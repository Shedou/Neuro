Stable Diffusion web UI Neuro v4.

Портативная сборка создана для упрощения запуска и использования Stable Diffusion web UI.

Пожалуйста, воздержитесь от использования данной сборки если вы не готовы взять ответственность за любые возможные и невозможные последствия!
Внимание! Нейросеть может создавать неприемлемые изображения в зависимости от используемой модели и действий пользователя!

- Описание:
Не умеете рисовать, но очень хочется? Позвольте это сделать приложению Stable Diffusion!
Нейросеть нарисует всё что пожелаете, конечно же в пределах возможностей используемой модели...
Для генерации изображения достаточно написать текст в поле ввода и нажать кнопку "Generate",
но есть один нюанс, нейросеть понимает только слова на английском языке, впрочем,
никто не запрещает использовать переводчик отдельно если с английским языком всё очень плохо...

- Установка:
        Распаковать, установить модели и использовать.

- Использование:
    WebUI-CPU.bat                Использовать ЦП для работы (Режим работы без видеокарты).
    
    WebUI-LowVRAM-XFormers.bat   Для видеокарт с 2-6 GB VRAM. РЕКОМЕНДУЕМЫЙ вариант запуска если у видеокарты менее 8 GB видеопамяти!
    WebUI-LowVRAM.bat            Для видеокарт с 2-6 GB VRAM.
    
    WebUI-MedVRAM-XFormers.bat   Для видеокарт с 8+ GB VRAM. При нехватке памяти работает хуже чем LowVRAM. Запуск с оптимизациями Xformers.
    WebUI-MedVRAM.bat            Для видеокарт с 8+ GB VRAM. При нехватке памяти работает хуже чем LowVRAM.
    
    WebUI.bat                    Обычный запуск без оптимизаций.
    WebUI-XFormers.bat           Обычный запуск с оптимизациями Xformers. Может повысить производительность.
    WebUI-NoFP16.bat             Обычный запуск без оптимизаций. Использовать FP32 вместо FP16. Увеличивает расход памяти.
    
    WebUI-Force XFormers.bat     Принудительно использовать xformers, не факт, что это нужно.

Сборка Stable Diffusion web UI Neuro v4 проверялась с видеокартами:
- GeForce GT 1030 2GB (R5 3500X, 32GB RAM) - оптимально для моделей SD в разрешении до 768x768.
- GeForce GTX 1650 Mobile 4GB (i5-10300H, 8GB) - оптимально для SD в разрешении до 1024х1024. 8 ГБ системной ОЗУ снижали эффективнось работы.
- GeForce RTX 3060 Mobile 6GB (i7-10875H, 64GB) - оптимально для SD/SDXL в разрешении до 1920x1200.

- Минимальные системные требования:
        ОС: 64 разрядная Microsoft Windows 10 / 11
        ЦП: 64 разрядный процессор, 2 ядра
        ОЗУ: 8 ГБ
        Видеокарта: GeForce GTX 900 серия или новее*
        Видеопамять: 2 ГБ
        Браузер: Microsoft Edge или лучше
* - Сборка не тестировалась на видеокартах серии GTX 900.
